Working with Colors

Tailwind CSS supports over 220 colors by default — including all base colors, their shades, and utility variants.

In tailwind, colors shades are ranges form 50 to 950, and some of the colors do not have shades in tailwind css like: black , white, transparent

In CSS (and Tailwind), an outline is a line drawn outside the element’s border to highlight or indicate focus. A border is a line that is drawn on the edge of the element’s box — it is part of the box model and affects layout (e.g., width/height).
